Bootstrap Popover Picker

Generic jQuery plugin template for building pickers using Bootstrap popovers, fully customizable with a powerful base API.

Demo featuring a Font Awesome icon picker (bundled with the library)

$('.icp-1, .icp-2').picker();
$('.icp-3').picker({
    title: 'With custom options',
    selectableItems: ['github', 'heart', 'html5', 'css3'],
    selectedCustomClass: 'label label-success',
    displayButtons: true,
    placement:'topRight',
    // note that this is ignored cause we have an accept button:
    hideOnPick:true,
    //... (see the source code of this page)
});
